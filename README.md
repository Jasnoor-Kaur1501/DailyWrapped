# DailyWrapped

Spotify Wrapped, but every single day.

Log the songs you played today, pick a mood for each one, and generate a beautiful wrapped-style card with stats, mood breakdown, and your full tracklist.

## Live Demo

🔗 [jasnoor-kaur1501.github.io/DailyWrapped](https://jasnoor-kaur1501.github.io/DailyWrapped)

## What it does

- Log songs with artist, mood tag, and a color
- Generate a wrapped card showing your top song, mood breakdown, unique artists, and vibe rating
- 6 mood tags — Hyped, Emotional, Vibing, Chill, Sad, Midnight
- Dynamic hero card background built from your song colors
- Copy your wrap to share

## Vibe Ratings

| Songs logged | Vibe |
|---|---|
| 3–5 | 🌱 warming up |
| 6–9 | 🔥 on one today |
| 10–14 | 🌀 unhinged (affectionate) |
| 15+ | 👑 chronically online |

## Design

Bricolage Grotesque + Fraunces italic · Deep charcoal, purple-to-orange gradient accents · Spotify Wrapped energy meets scrapbook aesthetic. Completely different from Days 01, 02, and 03.

## Tech

HTML · CSS · Vanilla JS · No dependencies · No API · GitHub Pages

## Run locally

Download or clone → open `index.html` in any browser. Done.
